# bubble-online-store
Minimalistic eCommerce store built on Bubble.io that allows users to manage products, filter through a live storefront, add items to a cart, and process orders. 

**Feature List**

- Inventory Management
- Storefront Display
- Cart Functionality
- Price & Keyword Filters
- Order Management
